# Project_EX
A custom renderer built with C++ and OpenGL, complete with a real-time editor UI.

It features a modern PBR workflow, image-based lighting (IBL) with clear coat support, and both forward and deferred rendering pipelines.

Additional systems such as advanced rendering techniques and scene serialization are also demonstrated.
